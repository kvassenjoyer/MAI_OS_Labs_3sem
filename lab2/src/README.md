# Лабораторная работа №2

## Условие:
Вариант 18.	Найти образец в строке наивным алгоритмом

## Сборка лабы:
```
gcc npsaive_pattern_search.c -o nps
```

## Запуск лабы:
```
./nps n
```
, где n - количество потоков

## Отладка лабы с помощью strace:
```
strace -f ./nps n
```
, где n - количество потоков
